### Users features:
- Register;
- Login;
- Password change;
- Profile info;

### Shipper features:
- Delete account;
- Create loads;
- View created loads;
- Update loads with status 'NEW';
- Delete loads with status 'NEW';
- Post a load;
- View shipping info.

### Driver features:
- Add trucks;
- View created trucks;
- Assign 'In Service' status to truck;
- Update trucks info;
- Delete trucks;
- View assigned to the driver load;
- Interact with assigned loads (pick up and deliver).
